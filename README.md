# 📖 Hello React Front End <a id="about-project"></a>

**Hello React Front End** is the front-end of an application that consumes an API and displays a random greeting message.

## 🎨 Back-end Repo

This project is working with this Back-end [GitHub repository](https://github.com/Luffytaro22/hello-rails-back-end).

## 🛠 Built With <a id="built-with"></a>

### Tech Stack

* HTML5
* CSS3
* JavaScript
* React.js & Redux
* Vite

<!-- Features -->

### Key Features
* SPA with react router.
* Update the redux store with information of the API.
* Integrate Back-end and Front-end.

<!-- GETTING STARTED -->

## 💻 Getting Started <a id="getting-started"></a>

### Prerequisites
In order to run this project you need to have:

- A Text Editor.
- A Browser.

### Setup
To clone this repository to your desired folder, run the following command: <br>

```
git clone git@github.com:Luffytaro22/hello-react-front-end.git
```

### Install
```
 npm install
```

### Usage

Initialize it with <br>
```
npm run dev
```
